Mes projets

  • Service de configuration d'écrans digitaux

    Plus de détails Visiter le site

    Babi est un prototype fonctionnel d'entreprise d'écrans digitaux. Les restaurants, compagnies locales, et entrepreneurs pourront tous profiter de grand écrans afin de moderniser leur facade d'entreprise. Installez notre logiciel sur un Raspberry Pi, scannez le code QR affiché au redémarrage, et configurez votre écran sur notre Tableau de bord. Du menu du jour aux publicités, tout est affichable.

    Compétences utilisées :

    1. Base de données MongoDB pour informations utilisateurs (emails, mots de passe encryptés, etc)
    2. Pages de création de compte et de connexion
    3. Création d'un tableau de bord complet avec contrôle des appareils connectés
    4. Conception et intégration d'un logiciel compatible avec Ubuntu Server, Ubuntu Desktop, Ubuntu Core, Rasperry Pi OS et Arch Linux.
    5. Programmation du logiciel avec NodeJS et des scripts Bash, le tout s'exécutant dans un conteneur Docker sous Alpine Linux.

  • Service d'analyse de fichiers 3D

    Plus de détails Visiter le site

    Crewello est un API pouvant déterminer les dimensions, le poid, et le prix de différentes pièces 3D importées. Il peut également visualiser presque instantanément n'importe quel fichier dans un environnement 3D grâce à la librairie ThreeJS. Avec une quarantaine de fichiers supportés, ce logiciel est techniquement l'un des plus complet et compétitif sur le marché.

    Compétences utilisées :

    1. Librairies ThreeJS et CuraWASM
    2. NodeJS
    3. Hégergement sur Heroku
    4. Configuration de nom de domaine
